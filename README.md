# Pi-MG90S-Motor

Run command 
'''
 git clone https://github.com/tuj05689/Pi-MG90S-Motor
'''
Add permissions
'''
 chmod -Rf 777 Pi-MG90S-Motor
'''
run setup
'''
 sudo python setup.py install
'''
Connect the motor to the rasberry pi
'''
 Brown is ground
 Red is 3.3 Volts
 Orange is PWM control, Using Pin 13
'''
